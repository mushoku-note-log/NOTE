実際にnote以外のSNSへ投稿した（または投稿案として作成した）文章の記録です。

- `threads/`：Threadsへの投稿案・投稿文（本文にnoteのURLを直接貼らず、プロフィール欄経由で誘導する方針。詳細はPROJECT_RULES.md「Threads運用ルール」参照）

Threadsアカウントは2026年7月11日に停止したため、現在は新規投稿はありません。他のSNS（X、Instagram、Pinterest、YouTube、TikTok）については、投稿文の個別記録はまだ整備していません（今後追加予定）。
